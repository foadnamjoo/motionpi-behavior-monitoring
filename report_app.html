SSH Tunnel Guide (Mac)

Use this one-time tunnel setup each session, then run the report.

Quick check: keep your SSH tunnel terminal open while using this page.
  1. Open Terminal.
  2. Run this command (replace placeholders with your internal hosts/usernames): ssh -J YOUR_USER@YOUR_BASTION_HOST -L 27017:127.0.0.1:27017 YOUR_USER@YOUR_DB_HOST
  3. Complete your organization's authentication flow (password/MFA as required).
  4. Leave the terminal open and return here to click Run report.

Tip: You are connected when you see a remote shell prompt like [your_user@your_db_host ~]$.

If report shows "Could not reach the report server", start local server first (`python report_server.py`) and check your tunnel.
